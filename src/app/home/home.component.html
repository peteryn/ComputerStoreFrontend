Demo Explanation

User Registers

User enters email and password

User Login

Use logs in with email and password

Server Responds

Server verifies credentials and responds with JSON Web Token (JWT)

User accesses secure page

The user is trying to access a page that only a logged in user can access. The user must include the JWT in the request

Server Verifies JWT

If the JWT is valid, server responds with secured page details

User accesses service

The user can use the secured resource and perform other actions
